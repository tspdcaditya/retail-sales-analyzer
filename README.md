# Dynamic Retail Sales Analyzer

A Python desktop app that lets you upload any retail sales CSV/Excel file 
and instantly generates visual dashboards using Matplotlib.

## Features
- Auto-detects Sales, Date, Category, and Product columns
- Sales trend over time
- Category-wise sales comparison
- Top 5 & Top 15 product breakdown
- Sales distribution histogram
- Download all charts as PNG files

## Tech Stack
Python · Pandas · Matplotlib · Tkinter

## How to Run
1. Install dependencies: `pip install -r requirement.txt`
2. Run: `python app.py`
3. Upload any retail CSV file and click Analyze
